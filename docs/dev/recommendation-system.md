# Sistema de recomendação (design)

O Playlite tem a base para um sistema de recomendação com foco em:

- **Local-first** (privacidade e offline)
- **ML clássico** (simples, performático, explicável)
- **LLM opcional** apenas para explicar resultados (não decide ranking)

Para o design técnico completo, veja: `/recommendation-system` no repositório.
