# Playlist

A **Playlist** é onde você monta uma sequência de jogos para jogar.
Pense como uma fila: você escolhe e organiza os próximos títulos.

## O que dá pra fazer

- Adicionar jogos na playlist
- Reordenar itens (arrastar e soltar, quando disponível)
- Remover jogos da playlist

## Como usar bem

- Mantenha a playlist pequena (ex.: 5–10 jogos) para continuar prática.
- Use a playlist como um compromisso leve: “esses são os próximos”.

## Dicas

- Se você curte desafios, crie playlists temáticas (ex.: só indies, só RPG, só coop).
