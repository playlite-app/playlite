# Favorites

A tela **Favorites** mostra apenas os jogos que você marcou como favoritos.
Ela funciona como uma “lista curta” do que você mais gosta (ou do que quer priorizar).

## O que dá pra fazer

- Ver seus jogos favoritos em um só lugar
- Buscar por nome (quando disponível na UI)
- Remover um jogo dos favoritos

## Dicas

- Use favoritos como sinal de prioridade: “quero jogar isso em breve”.
- Se a lista estiver grande demais, mova alguns itens para a **Wishlist**.
