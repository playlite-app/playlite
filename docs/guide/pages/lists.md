# Listas e Organização

O Playlite oferece maneiras distintas de organizar o que você quer jogar.

![Playlist](/images/app-playlist.png)

Conheça as três e entenda a diferença entre elas:

## 1. Playlist (Sua Fila)

A **Playlist** é a sua ordem de jogatina. Pense nela como uma fila de "Próximos a Jogar".

- **Quando usar:** Para jogos que você já tem e quer jogar em breve.
- **Dica:** Mantenha esta lista curta (5 a 10 jogos) para não criar ansiedade. Use para focar no que importa agora.

## 2. Wishlist (Lista de Desejos)

A **Wishlist** é para jogos que você **ainda não tem** ou não pretende jogar agora.

- **Quando usar:** Viu um jogo interessante no *Trending*? Jogue para a Wishlist.
- **Objetivo:** Não esquecer de títulos interessantes para futuras promoções.

## 3. Favorites (Favoritos)

A página de **Favorites** é o seu "Hall da Fama".

- **Quando usar:** Para jogos que você ama, joga recorrentemente (como multiplayer) ou quer ter acesso imediato.
