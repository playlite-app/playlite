# Funcionalidades

> A ideia é que você continue com seu histórico local, independente de plataforma.

- Exportar/importar seus dados (JSON)

## Backup & Restore

- Preencher automaticamente parte dos dados (nome, capa, tempo jogado, etc.)
- Importar sua biblioteca da Steam via Steam Web API

## Importação da Steam (opcional)

- Usar como fonte de ideias para wishlist/biblioteca
- Descobrir jogos em destaque

## Trending

- Reorganizar itens para definir prioridades
- Montar uma fila/ordem de jogos para jogar

## Playlist

- Separar “descobertas” da sua biblioteca principal
- Manter uma lista de jogos que você quer jogar/ter no futuro

## Wishlist

- Acessar rapidamente os jogos que você quer priorizar
- Marcar jogos como favoritos

## Favoritos

- Registrar **tempo jogado** e **nota**
- Adicionar capa (opcional)
- Buscar por nome
- Adicionar/editar/remover jogos

## Biblioteca (Library)

O Playlite é um gerenciador de biblioteca de jogos **desktop** com foco em ser simples, rápido e **local-first**.
