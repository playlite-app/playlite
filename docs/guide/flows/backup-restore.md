# Fluxo: backup e restore (JSON)

Este fluxo ajuda a exportar/importar seus dados do Playlite.

## Backup (exportar)

1. Vá em **Settings**.
2. Localize a ação de **Backup / Exportar dados**.
3. Escolha um local para salvar o arquivo.
4. Confirme.

## Restore (importar)

1. Vá em **Settings**.
2. Localize a ação de **Restore / Importar dados**.
3. Selecione o arquivo de backup (JSON).
4. Confirme e aguarde.
5. Verifique em **Libraries** se os dados voltaram.

## Boas práticas

- Faça backup antes de grandes mudanças (importação, reorganização, etc.).
- Guarde o arquivo em um local seguro (ex.: pasta de backups, nuvem, pendrive).

## Se der problema

- Verifique se o arquivo é um JSON gerado pelo Playlite.
- Se o restore falhar, tente em uma cópia limpa do app (ou faça um backup do estado atual antes).
