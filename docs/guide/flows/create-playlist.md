# Fluxo: criar e organizar uma playlist

A playlist é uma forma prática de montar uma fila do que você quer jogar em seguida.

## Passo a passo

1. Vá em **Playlist**.
2. Clique em **Adicionar** (ou escolha um jogo na biblioteca e mande para a playlist, se a UI suportar).
3. Adicione alguns jogos.
4. Organize a ordem:
  - arraste e solte (quando disponível), ou
  - use os controles de mover para cima/baixo (se existir).
5. Remova itens quando não fizerem mais sentido.

## Dicas

- Mantenha uma playlist curta (5–10 jogos) para não virar “segunda wishlist”.
- Monte playlists temáticas: curto/rápido, história, coop, etc.
