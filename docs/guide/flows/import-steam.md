# Fluxo: importar biblioteca da Steam

Este fluxo te guia no caminho mais direto para trazer sua biblioteca da Steam para o Playlite.

## Pré-requisitos

- Ter um **SteamID (64-bit)**
- Ter uma **Steam Web API Key**
- Estar com internet no momento da importação

## Passo a passo

1. Abra o Playlite e vá em **Settings**.
2. Preencha:

- **SteamID**
- **API Key**

3. Clique na ação de **Importar biblioteca da Steam**.
4. Aguarde a conclusão.
5. Vá em **Libraries** e confira os jogos importados.

## O que esperar do resultado

- Os jogos são adicionados na sua biblioteca local.
- Alguns campos podem vir incompletos (por exemplo, gênero), e você pode ajustar depois.

## Se der problema

- Confira se SteamID e API Key estão corretos.
- Verifique as configurações de privacidade da conta Steam.
- Veja o **FAQ**: `/guide/faq`.
