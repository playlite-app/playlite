# Fluxo: adicionar um jogo na biblioteca

Este é o caminho mais rápido para colocar um jogo no Playlite manualmente.

## Passo a passo

1. Vá em **Libraries**.
2. Clique em **Adicionar jogo**.
3. Preencha o mínimo:

- **Nome**

4. (Opcional) Complete com:

- **Gênero**
- **Plataforma**
- **Capa (URL)**
- **Nota**
- **Tempo jogado**

5. Salve.
6. (Opcional) Abra os detalhes do jogo e ajuste o que quiser.

## Dicas

- Se você pretende jogar em breve, marque como **Favorite**.
- Use a **Wishlist** para jogos que você ainda não quer colocar na biblioteca.
