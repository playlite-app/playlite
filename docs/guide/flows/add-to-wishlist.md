# Fluxo: adicionar um jogo na wishlist

A wishlist é o lugar ideal para guardar jogos que você descobriu e quer lembrar depois.

## Passo a passo

1. Vá em **Wishlist**.
2. Clique em **Adicionar**.
3. Informe o nome do jogo (e o que mais fizer sentido, se a UI oferecer campos opcionais).
4. Salve.

## Dicas

- Use a wishlist para descobertas do **Trending**.
- Quando virar prioridade, você pode mover para a biblioteca (ou adicionar o jogo na **Libraries**).
