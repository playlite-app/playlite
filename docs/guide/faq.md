# FAQ

## A importação da Steam não funciona. O que verificar?

- Seu **SteamID** está no formato **64-bit**
- Sua **Steam Web API Key** é válida
- Seu perfil/biblioteca está visível (dependendo das configurações de privacidade)
- Você está com internet no momento da importação

## Por que alguns dados vêm incompletos (ex.: gênero)?

Algumas informações não vêm diretamente da API usada na importação. Nesses casos, o Playlite preenche o essencial e você
pode editar depois.

## Como faço backup/restore?

Se a sua versão do app tiver essas opções na UI, use as ações de backup/restauração para exportar/importar um JSON.
