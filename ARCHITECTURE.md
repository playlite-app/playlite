# Project Architecture

This document summarizes the project architecture and the repository structure. Autogenerated/config folders are
omitted.

## Overview

- **Frontend**: `src/` contains the React UI, hooks, and shared UI components.
- **Backend**: `src-tauri/` contains the Tauri Rust backend, data access, and integrations.
- **Data**: `data/` and `src-tauri/data/` store datasets and metadata used for recommendations and tagging.
- **Docs**: `docs/` holds guides and developer documentation.

## High-level flow

1. The UI calls Rust commands via Tauri IPC.
2. Rust services aggregate data from local storage and external APIs.
3. Results are cached and returned to the UI for rendering.
